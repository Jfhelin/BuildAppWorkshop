# Title
Workshop App Requirements

# Labels
requirements

# Body
Create a small internal productivity app that can be completed as a 90-minute prototype.

The app should help a user capture, review, and act on a lightweight set of work items.

Requirements:

- the app must have a clear home screen
- the app must support creating or reviewing at least one item type
- the app must show item status or progress
- the app must use sample or mock data only
- the app must work as a browser-based prototype

Non-requirements:

- authentication
- payment flows
- real production integrations
