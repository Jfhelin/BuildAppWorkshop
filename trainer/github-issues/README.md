# GitHub Issue Seeds

These markdown files are seed content for the live GitHub issues used in the workshop.

Before the session:

1. Create a GitHub repository from this workshop kit.
2. Publish each markdown file in this folder as a GitHub issue.
3. Apply the labels listed in each file.
4. Keep the issue titles aligned with the seed files so participants can reference them easily through GitHub MCP.

Recommended issue set:

- `Workshop App Requirements`
- `Workshop Acceptance Criteria`
- `Workshop Design Guidance`
- `Optional Stretch Goal`
