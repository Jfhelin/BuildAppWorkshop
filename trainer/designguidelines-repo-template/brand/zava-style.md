# Zava Style

This file defines the workshop Zava design language.

This is a workshop-defined brand direction inferred from the provided Zava demo references and logo samples. It should be treated as the source of truth for workshop branding unless replaced by more specific guidance.

## Design Intent

- trustworthy
- calm
- modern
- approachable
- clean and product-focused
- premium but not flashy

## Visual Traits

- light backgrounds and white surfaces
- strong spacing rhythm
- clear section hierarchy
- bright azure-blue accent usage in focused locations
- clear cards and panels
- clean, readable typography
- rounded corners and softened edges
- compact but breathable layouts

## Brand Shape

- default brand color is a vivid sky-to-azure blue
- logo can sit inside a rounded blue badge
- white or very light lettering is preferred on blue brand surfaces
- the logo should feel crisp, simple, and horizontal

## Product Feel

- resembles a polished self-service app more than a marketing page
- hero areas can exist, but they should support tasks, not dominate the screen
- account screens, sign-in surfaces, and product grids should feel orderly and lightweight

## Avoid

- noisy gradients
- glassmorphism
- playful cartoon illustration
- exaggerated animation
- dark-heavy dashboards unless specifically requested
- crowded interfaces
